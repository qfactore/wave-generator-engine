# X-Alpha Standard

This is the locked WGE-1 root Source Profile. Its seven session assignments,
authority references, calibration identity, channel convention, and exact-trust
surface are data. It is non-executable.

The authority defines Focus as a remappable role over logical channels `0–7`
but does not certify one default target. The locked LeverSet therefore preserves
the role bundle and explicitly leaves the target unresolved instead of guessing.
