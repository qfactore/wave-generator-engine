# WGE-3 Scope

WGE-3 generates deterministic plans and graphs, not audio. Session 1 receives
Baseline Mode through profile data and uses exact frozen motif references.
Focus Role target is supplied by the run and does not alter the locked profile.

There is no carrier waveform, waveform timeline, calibration application,
playback intensity application, Dense scheduler, Complex scheduler, renderer,
audio exporter, or playback exporter. WGE-4 will render the validated Session 1
plan into the first WAV. WGE-4 has not started.
