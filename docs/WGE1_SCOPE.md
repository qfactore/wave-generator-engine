# WGE-1 Scope

WGE-1 establishes configuration contracts, schemas, registries, deterministic
hashes, lifecycle validation, safe draft forking, inspection CLI commands, and
Run Request validation.

Complex Mode state scheduling is not implemented. There is no motif access,
scheduler, renderer, transform execution, generation, SessionPlan, RenderPlan,
WAV export, playback JSON export, or upload payload. WGE-2 has not started.
