# Calibration Model

The calibrated 100% reference multiplier is 1.1. Playback intensity is a
separate post-calibration scalar with a reference default of 0.80. Relative
amplitude relationships are preserved. Per-motif normalization, per-session
normalization, and a default limiter are blocked. Float64 intermediate handling
and a −3 dBFS ceiling are future validation policy only. 24-bit delivery is
provisional and not hardware-certified.
