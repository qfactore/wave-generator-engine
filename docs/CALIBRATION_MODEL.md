# Calibration Model

Calibration belongs to the Source Profile through
`x_alpha_reference_calibration_v1`. Validation resolves the 1.1 calibrated
100% reference multiplier, preservation of relative amplitudes, no per-motif or
per-session normalization, no default limiter, and fail-closed headroom.

No gain is applied in WGE-1. Playback default belongs to Delivery Preset
metadata and remains separate from calibration, motif amplitude, and Focus Role.
