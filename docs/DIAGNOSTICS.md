# Diagnostics

Diagnostics are reproduced entirely from plan metadata. Raw JSON and CSV back
18 PNG figures covering timelines, fixed-bin densities, channel activity,
Focus Role comparison, motif use and repetition, grammar use, transitions,
Pulse Pattern, continuation counts, packet spans, and relative gains.

Each density definition is explicit, such as event onsets per one-second bin.
Every graph is labelled “Diagnostic plan metadata only — no waveform rendered.”
No physiological, therapeutic, perceptual, or source-authority claim is made.
