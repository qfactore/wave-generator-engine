# Motif Integrity

Whole-file SHA-256 gates access. Per-motif SHA-256 follows the reviewed upstream
convention: UTF-8 dtype text, UTF-8 shape text, then contiguous native bytes.

Metrics include sample count, duration, extrema, sample peak, RMS, mean, DC
offset, and zero-crossing diagnostics where mathematically defined. They are
computed from finite values without changing source arrays and make no carrier,
physiological, or perceptual claim.
