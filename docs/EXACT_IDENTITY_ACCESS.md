# Exact Identity Access

Exact Identity Access is a direct immutable lookup. It is not a transform and
does not enter a configurable operation pipeline.

Every receipt records an empty operation list, exact bypass, no randomness, no
transform path, matching source/result dtype and shape, bitwise equality,
read-only state, archive and motif hashes, and provenance references.

The API accepts no gain, resampling, time-scaling, transform, or operation
parameters. No independent carrier-frequency control exists.
