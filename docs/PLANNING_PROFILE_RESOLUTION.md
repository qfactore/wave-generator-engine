# Planning Profile Resolution

The resolver reads the locked profile, delivery preset, Interchange grammar,
fixtures, policies, and decision registry. Tier 1 vocabulary and safety rules
are binding. Tier 2 numeric guidance is copied into an immutable run snapshot
with value, unit, tier, artifact, source field, and diagnostic binding status.

Tier 3 and Tier 4 inputs are excluded. Missing runtime numbers use explicitly
labelled identity-safe provisional defaults or fail closed.
