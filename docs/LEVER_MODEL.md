# Lever Model

Lever definitions are registry data; LeverSets hold values; Basic and Advanced
are presentation views over that same registry.

Focus Role is a role bundle over logical channels `0–7`. Density emphasis moves
with the role, while global playback intensity and render calibration do not.
Authority certifies remapping but does not certify one standard default target,
so the locked profile records the target as unresolved rather than guessing.

Pulse Pattern candidates are future Advanced metadata with no invented ranges.
Motif time scaling is experimental, certificate-gated, range-un-certified, and
non-executable. Exact mode exposes no carrier-frequency lever. Calibration is a
locked Source Profile authority reference; playback default is delivery metadata.
