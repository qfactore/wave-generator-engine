# Event Plan

Every event records integer onset and duration samples, exclusive end position,
packet and unit identity, Pulse Pattern role, logical channel, Focus/support
role, exact motif ID and hash, source order, neutral relative gain, deterministic
selection trace, and authority references.

No waveform samples, transforms, calibration multiplier, playback intensity,
normalization, resampling, or time scaling appear. Headroom remains
`not_certified_without_waveform_render_and_overlap_sum`.
