# Authority Tiers

- Tier 0: immutable frozen authority.
- Tier 1: binding engine contracts.
- Tier 2: validated guidance and comparison references.
- Tier 3: provisional or reference research.
- Tier 4: archived or superseded material.

Tier 2 values are not universal production constants. Tier 3 is not certified.
Tier 4 is not current authority. Equal-tier conflicts fail closed.
